# These files help cross-reference potential units and quanitykinds. 

- RelatedQKs.csv - EDI QKs and the existing QUDT QKs that share the same dimension vector
- Relatedunits.csv - proposed EDI units with existing QUDT Units that share the same dimension vector
- RelatedQKs_summary.csv - count of how many existing QUDT QKs share the same dimension vector (that is to say, the number of lines in RelatedQKs.csv that have that dimension vector).
- RelatedUnits_summary.csv - count of how many existing QUDT Units share the same dimension vector (that is to say, the number of lines in Relatedunits.csv that have that dimension vector).
